On this page I introduce the different projects that made it possible for me to create this tool.

At the beginning of 2013 I started learning about elasticsearch. Since than I have done multiple projects for customers and I have given multiple trainings. I found out about repetitive tasks that were dealing with tools to maintain your elacticsearch nodes and the cluster. Therefore I decided to create a tool to help make these tasks easier to perform.

In the nearby future I am planning on creating more features related to configure your cluster an indexes.

If you find bugs or have ideas for improvements, please use github.

https://github.com/jettro/maintain-elastic

Dropwizard

website

Dropwizard is a Java framework for developing ops-friendly, high-performance, RESTful web services. I use it to create the REST api and to interact with the java driver of elasticearch.

AngularJS

website

AngularJS is the MVC framework for the client. The AngularJS client talks to my own REST api.

Twitter bootstrap

website

Twitter bootstrap is started as a pure css/html library to enable developers to create nice looking websites without to much effort. Nowadays a lot of (javascript) components are available that make life even easier.

UI Bootstrap

website

This is the glue between twitter bootstrap and angularjs. This library provides a number of directive that enable developers to easily create rich client side components.
